# spatial-analysis
The project was developed for university course _Spatial databases_. The purpose of this project was to analyse an ocean 
ships logs from 1750-1850. 

Analysis was done using R language with various analysis libraries. Analysis consists of, among others:
* generating variograms 
* prediction of wind strength[Kriging]
* visualizing data
